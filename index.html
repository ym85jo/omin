운동 시작 하기 전

  • 선수 추가: 왼쪽에 이름을 한 줄씩 입력 후 '선수 추가' 클릭
  • 팀 배정: 미배정 선수 또는 팀에서 드래그로 이동 (최대 4팀)
  • 랜덤 배정: 'Random 배정' 버튼 클릭 시 대기자 명단의 모든 선수가 선택한 팀 수에 맞게 무작위로 팀에 배정
  • 팀 색상: 팀 이름 클릭 시 색상(기본→노랑→주황) 변경
  • 결과: '결과' 클릭 시 현재 팀 상태를 표로 확인
  • 명단제외: 제외할 선수를 '명단제외' 영역으로 드래그하면 명단에서 삭제됩니다.
  • 초기화/저장: '초기화'로 모든 데이터 삭제, 변경사항은 자동 저장
