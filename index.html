운동 시작 하기 전

1. 선수 추가하기

  • 왼쪽 텍스트 영역에 참가할 선수들의 이름을 한 줄에 한 명씩 입력합니다.
  • '선수 추가' 버튼을 클릭하면 입력한 선수들이 미배정 선수 목록에 추가됩니다.
  • 이미 추가된 선수는 중복으로 추가되지 않습니다.

2. 팀 배정하기

  • 미배정 선수 목록에서 선수를 드래그하여 원하는 팀으로 이동시킵니다.
  • 팀 간에도 선수를 드래그하여 자유롭게 이동할 수 있습니다.
  • 각 팀은 최대 4개까지 생성되며, 각 팀의 선수 수는 제한이 없습니다.
  • 선수를 다시 미배정 목록으로 이동시키려면 '대기자 명단'으로 드래그하세요.

3. 팀 조끼 색상 설정

  • 팀 제목을 클릭하여 조끼 색상을 변경할 수 있습니다.
  • 클릭 순서: 기본 → 노란색(조끼) → 주황색(조끼) → 기본
  • 설정된 조끼 색상은 결과 보기에 자동으로 반영됩니다.
  • 각 팀의 조끼 색상은 독립적으로 설정 가능합니다.

4. 팀 배정 결과 보기

  • '결과' 버튼을 클릭하면 현재 팀 배정 상태를 테이블로 확인할 수 있습니다.
  • 각 팀의 선수들이 정렬되어 표시되며, 설정된 조끼 색상이 반영됩니다.
  • 모달 창 외부를 클릭하거나 X 버튼을 클릭하여 결과 창을 닫을 수 있습니다.
  • 결과는 실시간으로 업데이트되며, 팀 배정이 변경될 때마다 새로운 결과를 확인할 수 있습니다.

5. 데이터 관리

  • 모든 변경사항은 자동으로 저장되어 페이지를 새로고침해도 유지됩니다.
  • '초기화' 버튼을 클릭하면 모든 팀 배정과 선수 데이터가 삭제됩니다.
  • 초기화 전에 확인 메시지가 표시되며, 실수로 초기화하는 것을 방지합니다.
  • 브라우저의 로컬 스토리지를 사용하여 데이터를 저장하므로, 브라우저 캐시를 삭제하면 데이터가 초기화됩니다.

6. 팁

  • 팀 배정 시 드래그 앤 드롭이 잘 작동하지 않는다면, 브라우저를 새로고침해보세요.
  • 모바일 기기에서는 터치 드래그를 지원하지 않을 수 있습니다.
  • 최신 버전의 Chrome, Firefox, Safari, Edge 브라우저에서 최적의 성능을 보장합니다.
